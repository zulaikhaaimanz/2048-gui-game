# 2048 GUI Game

## Project Overview

This project is a graphical implementation of the classic 2048 puzzle game developed using Python and Tkinter. The game provides an interactive user interface where players combine numbered tiles by sliding them in different directions to reach the 2048 tile.

The project was created to strengthen concepts of Python programming, GUI development, event handling, and game logic implementation.

## Problem Statement

Traditional console-based games provide limited user interaction and visual experience. The objective of this project was to create a user-friendly graphical version of the 2048 game with an engaging interface and smooth gameplay mechanics.

## Working of the Project

1. The game starts with a grid containing randomly generated numbered tiles.

2. Players use keyboard arrow keys to move tiles in four directions:
   - Up
   - Down
   - Left
   - Right

3. When two tiles with the same value collide, they merge into one tile with double the value.

Example:

2 + 2 → 4

4 + 4 → 8

8 + 8 → 16

4. After each move, a new tile is generated at a random empty location.

5. The game continues until:
   - A player reaches 2048
   OR
   - No valid moves remain
